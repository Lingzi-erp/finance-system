# 财务管理系统

一个现代化的企业财务管理桌面应用，支持商品管理、订单处理、库存管理、财务结算等功能。

## ✨ 功能特性

- **商品管理** - 商品信息、分类、规格、包装规格管理
- **客商管理** - 客户/供应商信息管理
- **订单处理** - 采购单、销售单创建与管理
- **库存管理** - 实时库存查询、库存流水
- **财务结算** - 收付款管理、账款往来
- **数据统计** - 交易分析、资金流向、账龄分析
- **系统维护** - 数据备份、恢复、演示数据管理
- **自动更新** - 支持检测新版本并自动更新

## 🖥️ 系统要求

- Windows 10/11 (64位)
- 500MB 可用磁盘空间

## 📦 安装

1. 从 [Releases](https://github.com/Lingzi-erp/finance-system/releases) 下载最新版本
2. 运行安装程序
3. 按提示完成安装

## 🔧 技术栈

- **前端**: Next.js 15, React 18, TailwindCSS
- **后端**: Python FastAPI, SQLAlchemy
- **桌面**: Electron 28
- **数据库**: SQLite
